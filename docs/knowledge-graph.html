MCP Tools

Knowledge Graph Tools

Enhanced
search
대화 검색 결과에 관련 지식 컨텍스트(facts, ontology, relations)를 자동으로 부착하여 RAG 기반 응답 지원
Enhanced
search_facts
Fact 검색 시 온톨로지 분류(Domain/Category)와 1-hop 관계 그래프를 자동으로 포함하여 표시
New
trace_fact
지식그래프의 fact가 어떤 대화에서 추출되었는지 역추적. Revision history와 그래프 관계 포함
New
graph_stats
지식그래프 현황 통계: 전체 facts, domains, categories, relations 카운트 및 분포
New
cross_project_insights
다른 프로젝트의 유사한 기술 결정을 조회. 프로젝트 간 지식 전이로 반복 실수 방지
New
explore_graph
시드 fact에서 최대 3-hop 깊이까지 연결된 지식을 탐색. 간접 관련 결정까지 발견
Use Cases

활용 시나리오

🚀

새 프로젝트 시작

cross_project_insights로 기존 프로젝트에서 유사한 기술 스택 결정, 아키텍처 패턴을 참고하여 빠르게 부트스트랩

🏗

아키텍처 결정

explore_graph로 과거 결정의 연쇄 영향(A→B→C)을 파악하고, 새 결정이 기존 지식과 어떻게 연결되는지 확인

🔍

문맥 있는 검색

search가 자동으로 관련 facts와 ontology 컨텍스트를 부착하여, 단순 키워드 매칭이 아닌 지식 기반 답변 제공

📜

결정 근거 추적

trace_fact로 "왜 이렇게 결정했는지" 원본 대화를 복원하고, revision history로 결정의 진화 과정 확인

📊

지식 관리

graph_stats로 축적된 지식의 규모, 도메인 분포, 관계 유형을 모니터링하여 지식 체계의 건강도 파악
